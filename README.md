cnbeta-rss
==========

 - cnbeta 全文 + 热门评论 feed 输出 

 - 基于 nodejs , mongoDb  性能强劲  

 - 在 2 分钟内完成与 cnbeta.com 新内容的同步


demo : 
==========

 - http://cnbeta.cnodejs.net    NAE 云引擎
